# AGENTS.md — Правила работы над проектом Wavepeak

## Проект
Wavepeak — платформа с онлайн аудио инструментами (конвертеры, обрезка, усилитель громкости и т.д.).
Сайт: https://wavepeak.app
Репо: https://github.com/backinblaaaaaaack-sys/wavepeak

## Стек
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui — основная библиотека компонентов, использовать везде где возможно
- FFmpeg.wasm — обработка аудио прямо в браузере, без сервера

---

## Правила работы

### Общие
1. Отвечай ТОЛЬКО на русском языке
2. Никогда не соглашайся сразу — обсуждай, задавай уточняющие вопросы если задача не ясна
3. Думай перед ответом, будь уверен в решении на 100%
4. Никогда не импровизируй — если задача не ясна, остановись и переспроси
5. Объясняй простыми словами что делаешь — я учусь разработке

### Перед началом любой задачи
1. Изучи папку /memory-bank если она есть
2. Найди проблемный код в базе ПЕРЕД тем как что-то писать
3. Сформулируй чёткий план и покажи его мне
4. Жди моего разрешения перед любыми изменениями

### Формат ответа при баге или задаче
**ПРОБЛЕМА:** что именно не так
**ГДЕ ИМЕННО:** файл и строка
**КАК БУДЕМ РЕШАТЬ:** конкретные шаги
**ЧТО ИЗМЕНИТСЯ ПОСЛЕ:** результат
**КАК ПРОТЕСТИРОВАТЬ:** конкретные действия для проверки
**МОГУ ЛИ Я НАЧАТЬ ИСПРАВЛЕНИЕ?**

### Код
- Только минимальные и точные изменения — ничего лишнего
- Не дублируй код — выноси общее в отдельные модули
- Используй shadcn/ui компоненты везде где возможно
- Современный чистый стиль кода (2025, уровень Google/Apple)
- После каждого изменения — краткий отчёт: что поменялось, почему, какие файлы затронуты
- Разбивай задачи на мелкие понятные шаги

### Дизайн
- Тёмная тема по умолчанию
- Минималистичный, профессиональный SaaS стиль
- Все UI компоненты через shadcn/ui

### После каждой задачи
- Обновляй /memory-bank со всеми изменениями
